# Projeto-Conecta-Geracoes
Sistema para gerenciamento de currículos e feedbacks de recrutamento, desenvolvido em parceria com alunos de Administração como parte de um projeto acadêmico interdisciplinar. Visa aplicar da minha parte conceitos de engenharia de software, modelagem de banco de dados, persistencia de dados e desenvolvimento back e front-end.

# Processo de desenvolvimento do sistema:

Para esse projeto, escolhi tecnologias e modelos de desenvolvimento que agilizariam a implementação da aplicação.
Foram aplicadas conceitos aprendidos no curso de Ciência da Computação, o qual eu sou discente. Para isso, apliquei métodos de Banco de Dados e engenharia de software.
Foi escolhida um modelo de aplicação MVP, onde agrego o máximo de valor possível a aplicação utilizando o mínimo de "recursos".

Então, separei o desenvolvimento do app em partes, estruturação do banco de dados, desenvolvimento da API e por fim implementação do front-end (Interface Gráfica do app.)
